**Endpoints:**
```http
POST   /api/auth/register/          # Register a new user   ✔️
POST   /api/auth/token/             # Login, returns JWT    ✔️
POST   /api/auth/token/refresh/     # Refresh JWT           ✔️
GET    /api/auth/me/                # Get current user info ✔️
PUT    /api/auth/me/                # Update user profile   
````
